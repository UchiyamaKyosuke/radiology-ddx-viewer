# Radiology DDX Viewer

Static iPhone-friendly viewer for Radiology DDX `.rddx` packs.

## Use

1. Publish this repository with GitHub Pages.
2. Export `radiology-ddx-pack.rddx` from the PC admin app.
3. Copy the `.rddx` file to OneDrive.
4. Open the GitHub Pages URL on iPhone.
5. Tap `.rddx を選択` and choose the file from the iOS Files app or OneDrive provider.

Disease data is not stored in this repository. It is imported locally into the browser storage on the iPhone.
